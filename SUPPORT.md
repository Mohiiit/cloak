# Support

## Where to Get Help
- Feature requests and bug reports: GitHub Issues
- Security concerns: see `SECURITY.md`

## What to Include in Support Requests
- Environment (OS, Node version, package/workspace)
- Exact command(s) run
- Expected behavior vs actual behavior
- Relevant logs/error messages (redacted)
- Minimal reproduction steps

## Response Expectations
This is a maintainer-driven project. Response times may vary.
Critical security issues are prioritized.
